Settings

CORS Proxy

Tidal's API does not allow direct browser requests from external origins. A CORS proxy forwards your requests to Tidal and adds the necessary headers. The default uses corsproxy.io. Leave empty to attempt direct requests (may fail).

About

tiddl-web is the browser-based front-end for tiddl, the open-source Tidal downloader.

Authentication tokens are stored in your browser's localStorage and never leave your device.

For best results, install the CLI version which supports full quality, metadata, and format conversion via ffmpeg.
